# Content OS - Task Plan

**Goal:** One configurable writing system that outputs any format, in your voice, without sounding like AI.

**Project:** Content OS
**Started:** 2026-02-02
**Status:** `planning`

---

## Vision

**Not 30 skills. One writing engine.**

```
┌─────────────────────────────────────────────┐
│              YOUR VOICE/STYLE               │
│    (captured once, applied everywhere)      │
├─────────────────────────────────────────────┤
│              FRAMEWORK LAYER                │
│   (templates, structures, proven formulas)  │
├─────────────────────────────────────────────┤
│              FORMAT LAYER                   │
│  (newsletter, thread, carousel, script...)  │
└─────────────────────────────────────────────┘
```

**The Hello World Moment:**
1. Clone repo
2. Run voice capture wizard (3-5 writing samples)
3. Feed it source material
4. Pick a format
5. Get output that sounds like YOU
