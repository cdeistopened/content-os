## When to Reference

- **Phase 1 (Source Gathering)**: Quick review of Identity Framework to ground yourself
- **Phase 2 (Checkpoint 1)**: Reference Subject Line Guide for options
- **Phase 3 (Writing)**: Consult Style Examples for voice consistency
- **Phase 4 (URL Matching)**: No references typically needed
- **Phase 5 (QA)**: Compare against Style Guide and Identity Framework

## Progressive Disclosure Principle

Per Anthropic's skills protocol:
- **SKILL.md**: Core workflow instructions (~4,000 words)
- **References**: Loaded as needed during specific phases
- **Examples**: Consulted for structure and tone guidance

This keeps the main SKILL.md focused while providing depth when required.
