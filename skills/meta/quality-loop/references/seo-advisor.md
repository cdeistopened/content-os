# SEO Advisor - Expanded Reference

Advisory only - doesn't block publication. Provides recommendations for search visibility.

## Keyword Placement

| Location | Importance |
|----------|------------|
| H1 title | Critical |
| First 100 words | Critical |
| 2-3 H2 headings | Important |
| Meta title (50-60 chars) | Critical |
| Meta description (150-160 chars) | Important |
| URL slug | Important |

**Density:** 1-2% (roughly 1 mention per 100-200 words)

---

## Links

### Internal (Target: 5+)

- Link to relevant blog posts, podcasts, hub pages
- Use descriptive anchor text (not "click here")
- Distribute throughout content

### External (Target: 2-3)

- Link to authoritative sources
- Support statistics and factual claims
- Link to books/resources mentioned

---

## Featured Snippets

**Definition snippets:** 40-50 word direct answer right after the relevant H2

**List snippets:** Numbered list immediately after "How to X" heading

**FAQ snippets:** H3 with exact question, followed by 40-60 word answer

---

## Meta Elements

**Title:** Include primary keyword, compelling, under 60 chars

**Description:** Include keyword, clear value prop, 150-160 chars

**URL:** Clean, includes keyword, lowercase-with-hyphens
