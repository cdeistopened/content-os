Patterns: Normalize, Stop + Complaint, Observations, Rules, Pop Culture, Mock Instructions, Wordplay, Questions, Aspirational, Calendar, Struggles, Values.

## Loading Strategy

**For LinkedIn posts:**
1. Determine job-to-be-done (engagement, story, list, contrarian, authority, community)
2. Load ONE category file from `linkedin/`
3. Match concept to 5-10 templates
4. Select best 1-2 for execution

**For X/Twitter:**
1. Load `templates/one-liners.md` for short takes
2. Or `templates/post-structures.md` for threads
3. Reference `platforms/x-twitter.md` for heuristics

**For Facebook:**
1. Load `templates/one-liners.md` for question posts
2. Or `linkedin/engagement.md` for engagement formats
3. Reference `platforms/facebook.md` for heuristics

**For multiplying winners:**
1. Load `methods/proliferation.md`
2. Apply SCAMPER for 7 variations
3. Apply Human Desires for 8 more
4. Total: 17+ variations from 1 post
