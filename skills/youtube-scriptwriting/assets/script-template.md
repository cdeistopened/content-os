## Pre-Publish Checklist

### Hook Commandments
- [ ] Alignment: Visual + spoken + text hooks match
- [ ] Speed: Value promised in first 3 seconds
- [ ] Clarity: Topic is unmistakable
- [ ] Curiosity: Opens a question they must answer

### Story Flow
- [ ] One clear through-line
- [ ] No unnecessary tangents
- [ ] Logical progression beginning → end

### Comprehension
- [ ] 6th grade vocabulary
- [ ] Short, punchy sentences
- [ ] Active voice
- [ ] Complex ideas restated simply

### Speed to Value
- [ ] Value in first 3 seconds
- [ ] Rehooks at proper intervals
- [ ] Each section has payoff
- [ ] Outro satisfies or extends

### Emotion
- [ ] Target emotion achieved
- [ ] Emotional variety throughout
